# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |


## Reporting a Vulnerability

Falls du eine Sicherheitslücke entdeckst, melde sie bitte **nicht** als öffentliches Issue.

Erstelle stattdessen ein privates Issue oder kontaktiere den Repository-Inhaber direkt über GitHub. Bitte beschreibe die Lücke so genau wie möglich, damit wir sie schnell beheben können.
