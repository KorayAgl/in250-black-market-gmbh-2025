# Verzeichnisstruktur

## docs
Enthält alle Markdown Dokumentationen des Projekts.

## src
Enthält den Quellcode der Anwendung.

## img
Enthält Bilder und grafische Ressourcen.

## archive
Enthält alte Dokumente und Unterlagen aus der Vergangenheit.

## secrets
Enthält geheime Dateien wie Passwörter und Konfigurationen.

## website
Enthält die Vorlage für die GitHub Pages Website.
